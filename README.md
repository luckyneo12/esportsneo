# EsportsNeo - Ultimate Esports Experience

A modern, responsive esports website built with Next.js, featuring 3D animations, interactive components, and social media integration.

- **Dark Theme**: Professional dark color scheme with red accents
- **Neon Effects**: Red neon glows and gradients matching esports aesthetics

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/luckyneo12/esportsneo.git
   cd esportsneo
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
